Vilka är våra 349 ledamöter i riksdagen?

Sveriges riksdag består av 349 ledamöter som ska representera det svenska folkets röst. För att du ska få lite inblick i vilka dessa viktiga politiker är finns kortfattad information om dem samlad på den här webbplatsen (sveriges-riksdagsledamoter.se)

Filtrera med hjälp av nedanstående alternativ på antingen parti eller vilken valkrets de representerar, eller på både och. Vill du filtrera på namn kan du även göra det, men måste först välja alla partier och/eller alla valkretsar.

Välj själv hur många ledamöter du vill se på varje sida i tabellen. Välj också om du vill sortera dem på namn, födelseår, parti eller valkrets - i stigande eller fallande ordning.

Vill du få fram mer detaljerad information om en ledamot (uppdrag, utbildning, tidigare anställningar mm), klickar du på personens namn. Vill du istället se bild och kortfattad information om alla dina sökträffar, klicka istället på knappen "Visa info" under tabellen. Ifrån den här vyn kan du också klicka fram en enskild ledamots detaljerade info (samma som att klicka på personens namn i tabellen).

Härifrån kommer datan

• Den mesta datan kommer ifrån Sveriges riksdags API med ledamotsdata. Information om detta API hittas här: https://www.riksdagen.se/sv/dokument-och-lagar/riksdagens-oppna-data/ledamoter/

• Viss data kommer också ifrån Wikipedias API och är alltså hämtade från respektive partis Wikipedia-sida. Information om detta API hittas här: https://sv.wikipedia.org/w/api.php
